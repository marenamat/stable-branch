# Contributing

Contributions are welcome via pull request.

When submitting a pull request, add your name to `CREDITS.md`. Any UTF-8 name is accepted.
If your name changes later, update it via a PR and add an entry to `.mailmap`.

Maintainers aim to review submissions within one month.

Content written by a person is credited to that person. Prose generated by an LLM is not
credited. Code generated by an LLM may be credited to the human who directed and reviewed it.

This policy applies to itself.
